# FunnelSense

Scalable ecommerce funnel analytics built with DuckDB, Parquet, Streamlit, and an executive-ready insight layer.

## Features
- Session-based funnel analysis
- Brand and category comparison
- Daily conversion trends
- Insight summaries
- Built from 67M+ ecommerce events

## Tech Stack
- Python
- DuckDB
- Parquet
- Streamlit
- Plotly

## Run locally
```bash
streamlit run FunnelSense.py
